# ADL Business Consulting Landing Page

Modern, clean, and responsive "Coming Soon" page designed to match the ADL Business Consulting brand identity.

## Setup Instructions
1. **Repository Files**: Upload `index.html`, `CNAME`, and your logo image to this repository.
2. **Logo Filename**: Ensure your uploaded logo is named exactly `logo.jpg`.
3. **Custom Domain**: The `CNAME` file is pre-configured for `www.adlbusinessconsulting.com`.

## Deployment
This site is optimized for GitHub Pages.
- Ensure your GoDaddy DNS records (A and CNAME) are pointing to GitHub.
- Toggle "Enforce HTTPS" in the GitHub Settings > Pages menu once verified.
